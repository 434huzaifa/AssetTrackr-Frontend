## To run locally

```
git clone https://github.com/434huzaifa/AssetTrackr-Frontend
```
```
cd AssetTrackr-Frontend
```
```
npm install
```
```
npm run dev
```

NB: you can replace the baseURL inside useAxios.js with the local backend url to run both locally.
